我愛卡提諾
==========

沒錯, 你沒看錯, 我愛卡提諾。

我只想看圖, 可是卡提諾的網站載入太慢, 一堆廣告跟不必要的資訊, 還要等圖片 lazy load, 所以寫了這個 iloveck101, 直接先把圖片都下載下來, 看圖快多了。

目前只支援 Mac, Linux 我不確定。

by 海總理 2012.12.02



如何安裝
==========

```bash
$ pip install iloveck101
```


如何使用
===========

```bash
$ iloveck101 [url]

```

for example, 你可以試試看思穎

```bash
$ iloveck101 http://ck101.com/thread-2876990-1-1.html
```
